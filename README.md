Fee Adder
========================

Accounting program

Installation
------------

### Debian-based systems

```bash
sudo apt install libgtk-3-dev build-essential unzip wget -y

wget https://github.com/orchardstreet/fee_adder/archive/refs/heads/master.zip

unzip master.zip

cd fee_adder-master

./compile
```

Usage
-----------

### Run 

```bash
./fee_adder
```
