Fee Addr
========================

Accounting program

Installation
------------

### Debian-based systems

```bash
sudo apt install libgtk-3-dev build-essential unzip wget -y

wget https://github.com/orchardstreet/fee_addr/archive/refs/heads/master.zip

unzip master.zip

cd fee_addr-master

./compile
```

Usage
-----------

### Run 

```bash
./fee_addr
```
